# Port Scanner em Python

Projeto de scanner de portas desenvolvido em Python para estudos de cibersegurança.



##  O que o projeto faz

- Verifica portas abertas (1–1024)
- Identifica serviços (HTTP, SSH, etc.)
- Realiza banner grabbing
- Faz scan de vários IPs (rede)
- Permite salvar os resultados em arquivo



##  Como rodar o projeto

1. Baixe ou clone o repositório

2. Abra a pasta no VS Code

3. Execute o arquivo:

python scanner.py



##  Estrutura do projeto

- scanner.py - arquivo principal (onde roda o programa)
- utils.py - funções auxiliares (scan de porta, banner, etc.)
- examples/ - exemplos de resultados
- .gitignore - arquivos ignorados pelo Git



##  Exemplo de saída

[+] Porta 80 aberta (HTTP)  
    Banner: Apache/2.4.41  

[+] Porta 22 aberta (SSH)
